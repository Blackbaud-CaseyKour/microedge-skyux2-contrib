The tree view component displays data in an expandable/collapsable tree (parent/child) view.

The array of data to display in the component. The boolean value which determines if the parent nodes are disabled. The boolean value which determines if the leaf node is the only vale selectable. The boolean value which determines if the action controls are displayed. The boolean value which determines if the nodes are selectable (checkboxes are displayed). The TemplateRef value of the content to be displayed for each node. The TemplateRef value of the dropdown to be displayed for each node.
